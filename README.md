# Installation

Your ```.emacs``` file now needs something like this:

     (add-to-list 'load-path "/path/to/tramp-adb")
     (require 'tramp-adb)

Ensure the path to the SDK directory is configured:

    (customize-variable 'tramp-adb-sdk-dir)  

# Usage 

    (find-file "/adb::/")
